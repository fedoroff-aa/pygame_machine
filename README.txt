Небольшая инструкция.

* Это гоночный 2D симулятор, где цель сбить как можно меньше машин и поддерживать уровень
  топлива и технического состояния, собирая различные награды по пути, наезжая на них

* управление происходит при помощи стрелок (вверх - увеличение скорости, вниз - уменьшение, влево/вправо - движение между полосами)

* для старта нужно нажать кнопку "пробел"

* при победе/проигрыше высвечивается сообщение и также нужно нажать пробел для нового раунда

* сбоку есть 3 счётчика:
	- первый отвечает за количество очков
	- второй отвечает за количество сбитых машин
	- третий отвечает за количество раундов
